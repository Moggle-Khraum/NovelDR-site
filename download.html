v3.1.0rev322

July 4, 2026 Update

    ✨ [ NEW ]

  • Added Margins with 3 presets: Compact, Comfortable, & Spacious.
  • Added reader backgrounds with 11 presets (solid & gradient): Parchment, Night, Forest, Ocean, Rose, Slate, Dusk, Dawn, Mist, Moss, Ember, or keep the default by choosing None which automatically switches back to the app's theme and font colors.
  • Added adaptive text colors to adjust automatically for perfect contrast.
  • Added an option to set any image from gallery as a custom reading background.
  • Custom background images are stored locally; they will not be backed up to any cloud storage.
  • Added a function when tapping the progress bar to jump anywhere in the chapter.
  • Added UI overhaul to make the TOC display the current reading progress in percentage.
  • Added a dynamic search function inside the TOC to quickly search chapters by title, displaying results while typing.

⚡ [ IMPROVEMENTS ]

  • Reworked the skipping logic to work just like the Update Tab.
  • Improved LNW Scraping by adding diagnostics and support for direct link downloads.
  • Reduced skip delay from 200ms to 50ms per chapter, increasing skipping speed further.
  • Reworked the Developer Profile modal display.
  • Reworked the Credits and Acknowledgements modal display.
  • Control panel now a floating bottom‑sheet overlay — no longer pushes content down.
  • Progress bar relocated to just above bottom navigation for better visibility.
  • TTS highlight map precomputed with useMemo — no longer recalculated inside every render.
  • Reading progress saved every 3 seconds via debounce instead of only on unmount.
  • TTS scroll now follows the highlighted sentence using paragraph onLayout y‑positions.
  • Auto‑scroll stops when TTS starts; TTS stops when auto‑scroll starts.
  • Background preferences now persist across sessions via reader_bg.json.
  • ImageBackground fills screen by width/height with resizeMode="cover".
  • Added splitIntoReadableParagraphs() to normalize \r\n, tabs, blank lines, and single‑newline chapter text.

🗑️ [ REMOVED / REPLACED ]

  • Removed dead forceTopRef code.
  • Replaced outer content <Text> wrapper with <View> and rendered each paragraph as its own sibling <Text> with marginBottom — spacing is now actually respected.

🐞 [ BUG FIXES ]

  • Fixed covers occasionally saving as corrupted or empty files.
  • Fixed cover file extension always being saved as .jpg regardless of source format.
  • Resolved rare crashes when switching chapters while TTS was active.
  • Corrected text color behavior on gradient backgrounds.
  • Fixed “Next Chapter” preloading sometimes failing on the first attempt.
  • Prevented TTS from continuing after chapter change or app backgrounding.
  • splitIntoSentences no longer splits on commas/semicolons — sentence-ending punctuation only (.!?…), preventing choppy TTS fragments.
  • Added much smarter paragraph and sentence detection that handles dialogue, punctuation, line breaks, and special characters (→, …, —) for a natural reading flow.
  • Previously loaded chapters are stored temporarily in cache to make switching chapters flow seamlessly.
  • Optimized scrolling and text rendering; old cache entries are automatically cleared after 30 minutes.
  • Improved the persistence settings of the reader so that font size, line spacing, margins, auto‑scroll speed, background choice, and TTS preferences are saved automatically.

♿ [ ACCESSIBILITY ]

  • Added haptic feedback for all major interactions.
  • Added visual feedback on press for all touch targets.
  • Long‑press interactions are clearly documented in the TTS guide (book button) modal.
  • Added a Scroll to Top [ ↑ ] button in the reader to scroll directly to the very start of the chapter.
